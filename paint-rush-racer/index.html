// Gameplay Programmer · Co-op

◈// feature_suspension.png
Physics Arcade Suspension

Raycast-based suspension using a non-linear spring curve - stiff near the rest length for stability, softer at extremes to absorb big bumps. Compression and rebound damping are tuned independently per wheel, giving that snappy arcade signature without parameters fighting each other.

◈// feature_drift.png
Reworked Drift and Paint Mechanics

Reworked original prototype's multiplier-based drift and boost paint systems with arcade-style handling and a capped, scaling speed boost implementation.

◈// feature_gamemanager.png
Game Manager & Race State

Central Game Manager handles spawn logic for AI and player karts, the race lifecycle, and a continuous ranking system tracking position in real time using track progress rather than raw world distance.

◈// feature_paint.png
Kart and Paint Prototypes

Prototyped a new kart variant with a reworked paint spray mechanic alongside several alternate shooting modes - directional burst, AoE splat, and trail-paint. Cut for scope before submission, but early prototyping validated the core territory loop and surfaced design issues weeks before they would have become critical.
